# Data Structures & Algorithms in JavaScript

Welcome to Data Structures & Algorithms in JavaScript.

This repository is organized as a step-by-step learning path to make hard concepts easier to understand and practice through code.

## What You Will Learn

### Technical

- Big O notation

### Data Structures

- Arrays
- Linked Lists
- Doubly Linked Lists
- Stacks & Queues
- Binary Trees
- Hash Tables
- Graphs
- Heaps

### Algorithms

- Dynamic Programming
- Sorting
  - Bubble Sort
  - Selection Sort
  - Insertion Sort
  - Merge Sort
  - Quick Sort
- Searching
  - Breadth First Search (BFS)
  - Depth First Search (DFS)
